GLOBAL HOTKEY ON WAYLAND:
The keyboard library does not work reliably on Wayland. Replace with a Unix domain socket approach. main.py opens /tmp/friday.sock on startup and listens for incoming messages via asyncio. A Hyprland bind writes to the socket.
Add to hyprland.conf:
bind = CTRL SHIFT, SPACE, exec, echo "trigger" | socat - UNIX-CONNECT:/tmp/friday.sock
Socket listener in main.py:
pythonimport asyncio, os

async def socket_listener(trigger_callback):
    sock_path = "/tmp/friday.sock"
    if os.path.exists(sock_path):
        os.remove(sock_path)
    server = await asyncio.start_unix_server(
        lambda r, w: trigger_callback(), sock_path
    )
    async with server:
        await server.serve_forever()
Install socat: sudo pacman -S socat

Code extraction in llm.py — strip markdown fences defensively:
pythonimport re

def extract_code(response: str) -> str:
    response = re.sub(r"```[\w]*\n?", "", response)
    response = re.sub(r"```", "", response)
    return response.strip()

Token awareness — in-memory tracker in llm.py:
pythonimport time

token_log = {"minute_start": time.time(), "tokens_used": 0}

def update_token_count(usage):
    now = time.time()
    if now - token_log["minute_start"] > 60:
        token_log["minute_start"] = now
        token_log["tokens_used"] = 0
    token_log["tokens_used"] += usage.prompt_tokens + usage.completion_tokens

def get_token_status():
    return token_log["tokens_used"]
Add voice command "token status" or "how many tokens" → Friday says aloud: "Used X tokens this minute out of 6000." No persistent logging, in-memory per session only.
